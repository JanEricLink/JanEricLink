# Hi, ich bin Jan Eric Link 👋

**Master-Student Advanced Precision Engineering (M.Sc.)** an der Hochschule Furtwangen — mit Fokus auf **Robotik, Präzisionssysteme** und **Künstliche Intelligenz**.

Ich komme aus der Mechatronik (dualer Bachelor bei der Marquardt GmbH / DHBW Stuttgart) und verbinde Ingenieurwissen mit Softwareentwicklung. Mich interessiert vor allem **Künstliche Intelligenz** — insbesondere die praktische **Implementierung von Retrieval Augmented Generation (RAG)**. Dazu habe ich bereits eigene Projekte umgesetzt: ein RAG-System von Grund auf aufgebaut (inkl. Datenpipelines und Vektordatenbank), Web-App Entwicklung (Frontend, Backend, Database) und einen eigenen Chatbot.

### 🔬 Woran ich gerade arbeite

- **Lokales RAG-System** für unternehmenseigene Dokumente — komplett ohne Cloud-Anbindung.
- Forschung: *Wie verbessert Retrieval Augmented Generation die Ergebnisse von KI-Systemen?*
